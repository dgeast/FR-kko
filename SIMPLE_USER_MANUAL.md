# 📱 얼굴인식 오프라인 AI - 간단 사용자 및 키오스크 설정 매뉴얼

이 앱은 인터넷이나 별도의 서버 없이 삼성 태블릿 기기 자체 연산만으로 동작하는 오프라인 AI 얼굴인식 앱입니다.

---

## 🔒 1. 삼성 태블릿 "키오스크 모드(화면 고정)" 설정 방법
무인 환경(시연, 키오스크)에서 사용자가 앱 밖으로 나가거나 인터넷 설정 등을 건드려 **얼굴 데이터가 증발하는 것을 방지**하기 위해 반드시 화면을 고정해야 합니다.

1. **'앱 고정' 기능 활성화**
   - 삼성 태블릿의 `⚙️ 설정(Settings)` 앱을 엽니다.
   - `보안 및 개인정보 보호` > `기타 보안 설정`으로 이동합니다.
   - `앱 고정(Pin App)` 메뉴를 찾아 활성화(켜기)합니다.
   - (선택 사항) '앱 고정 해제 시 화면 잠금' 옵션을 켜두면 다른 사람이 임의로 고정을 풀 때 비밀번호를 요구하게 됩니다.

2. **얼굴인식 앱을 화면에 고정하기**
   - 크롬 브라우저에서 PWA(바탕화면에 추가된 얼굴인식 앱 아이콘)를 실행합니다.
   - 태블릿 하단 네비게이션 바에서 `||| (최근 실행 앱)` 버튼을 누릅니다.
   - 화면 중앙 상단에 떠 있는 **앱 아이콘**을 길게 터치하거나 클릭합니다.
   - 나타나는 팝업 메뉴에서 **`이 앱 고정 (Pin this app)`**을 선택합니다.
   - 이제 태블릿은 얼굴인식 앱 외의 다른 화면으로 전환되지 않는 완벽한 키오스크(무인기기) 모드로 작동합니다.

> 💡 **앱 고정 해제 방법**: 하단 네비게이션 바의 `||| (최근 앱)` 버튼과 `< (뒤로가기)` 버튼을 동시에 길게 누르면 고정이 해제됩니다.

## 🧑‍💻 2. 간단 사용자 매뉴얼

### [1단계] 앱 접속 및 사용자 등록
1. 태블릿 바탕화면에 생성된 **'얼굴인식 AI'** 아이콘을 눌러 앱을 실행합니다.
2. 메인 화면에서 `👤 사용자 등록` 버튼을 클릭합니다.
3. 본인의 이름(또는 닉네임)을 입력합니다.
4. 이름 입력 후 자동으로 **얼굴 촬영 화면**이 켜집니다.
5. 밝은 곳에서 카메라를 정면으로 응시한 뒤 `📸 사진 촬영 및 등록` 버튼을 누릅니다. (등록 완료)

### [2단계] 얼굴 인식(인증) 및 로컬 대화
1. 메인 화면에서 `🔍 얼굴 인식` 버튼을 누릅니다.
2. 렌즈를 쳐다보고 `📸 촬영 및 인식`을 누르면 즉시 누구인지 판별하여 자동 로그인됩니다.
   - *주의: 빛이 너무 없거나 심한 역광에서는 인식이 안 될 수 있습니다.*
3. 로그인 후 챗봇 화면이 열리며, "안녕", "이름" 등의 단어를 입력해 오프라인 AI와 대화할 수 있습니다.

### [3단계] 3일 데이터 소멸(보안) 안내
* 본 앱에 등록된 얼굴 사진(생체정보 데이터)과 사용자 기록은 등록 시점으로부터 **정확히 72시간(3일) 후 자동 영구 파기**됩니다.
* 앱이 구동될 때마다 유효기간을 체크하므로 관리자가 수동으로 지울 필요 없이 안전하게 시연할 수 있습니다.
