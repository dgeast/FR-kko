# 📱 샤오미 폰(안드로이드)에서 테스트 가이드

**목표:** 같은 WiFi 네트워크에서 PC와 모바일 폰을 연결하여 테스트

---

## 📋 사전 확인 사항

- ✅ PC와 샤오미 폰이 같은 WiFi 네트워크에 연결되어 있는지 확인
- ✅ 모바일 브라우저 (Chrome, Safari 등)
- ✅ PC에서 백엔드 + 프론트엔드 실행 가능 상태

---

## 1️⃣ PC에서 백엔드 서버 시작

### Windows PowerShell / CMD 에서

```bash
# 폴더 이동
cd facial_recognition/src/backend

# 가상환경 활성화
venv\Scripts\activate

# 서버 시작
python app.py
```

✅ 출력: `Uvicorn running on http://0.0.0.0:8000`

> **중요:** `0.0.0.0`으로 바인딩되어야 네트워크에서 접근 가능합니다.

---

## 2️⃣ PC의 IP 주소 확인

### Windows 명령프롬프트 / PowerShell

```bash
# IPv4 주소 확인
ipconfig
```

출력 예시:
```
이더넷 어댑터 무선 네트워크:
   IPv4 주소............................: 192.168.1.100
```

**기억해두기:** `192.168.1.100` (본인 PC IP)

### 또는 Mac/Linux

```bash
ifconfig
```

---

## 3️⃣ 프론트엔드 서버 시작 (다른 터미널에서)

```bash
# 폴더 이동
cd facial_recognition/src/frontend

# 패키지 설치 (처음 1회만)
npm install

# 프론트엔드 시작
npm start
```

✅ 출력: `Compiled successfully!` 및 `webpack compiled`

---

## 4️⃣ 모바일(샤오미 폰)에서 접속

### 단계별 진행

**Step 1:** 모바일 브라우저 열기
- Chrome, Edge, Safari, 또는 기본 브라우저

**Step 2:** PC의 IP로 접속
```
http://192.168.1.100:3000
```

✅ 화면이 나타나면 성공!

---

## 5️⃣ 앱에서 백엔드 URL 설정

### 모바일 앱 실행 후

1. **⚙️ 설정 버튼 클릭**
   - 홈 화면의 마지막 버튼

2. **백엔드 URL 입력**
   - 현재 기본값: `http://localhost:8000`
   - 변경: `http://192.168.1.100:8000`

3. **✓ 연결 테스트 클릭**
   - ✅ 성공 메시지 확인

4. **저장 후 뒤로**

## ⚠️ 문제 해결

### Q: "연결할 수 없습니다" 오류
```
원인: 백엔드 서버가 실행되지 않음
해결: 터미널에서 python app.py 실행 확인
```

### Q: 페이지가 로드되지 않음
```
원인: 잘못된 IP 주소
해결: 
1. ipconfig 로 PC IP 다시 확인
2. 모바일 > 설정 > URL 수정
```

### Q: 웹캠이 작동하지 않음
```
원인: 모바일 브라우저 권한 미설정
해결: 
1. Chrome 설정 → 권한 → 카메라 허용
2. 새로고침 후 재시도
```

### Q: 얼굴을 인식하지 못함
```
원인: 조명 부족 또는 거리
해결:
1. 밝은 환경에서 시도
2. 카메라에서 약 30cm 거리 유지
3. 정면을 향해 촬영
```

### Q: "백엔드에 연결되었지만 기능 사용 불가"
```
원인: CORS 오류 또는 API 문제
해결:
1. 백엔드 터미널에서 에러 확인
2. PC 방화벽 확인 (포트 8000 허용)
```

## 🔧 Windows 방화벽 설정

포트 8000을 허용해야 할 경우:

1. **Windows Defender 방화벽** 열기
2. **고급 설정**
3. **인바운드 규칙** → **새 규칙**
4. 규칙 유형: **포트**
5. 프로토콜: **TCP**, 포트: **8000**
6. **허용** 선택 후 완료
