# 📱 삼성 태블릿 모바일 시연 매뉴얼

이 매뉴얼은 PC(서버)에서 구동되는 **얼굴인식 앱**을 같은 WiFi 네트워크에 연결된 **삼성 태블릿(안드로이드)**에서 시연하기 위한 완벽한 가이드입니다.

---

## 1️⃣ [PC] 시연용 서버 준비

PC 환경에서 백그라운드 스크립트를 사용하여 서버를 가동합니다.

1. **서버 구동 스크립트 실행**
   - 명령프롬프트(CMD) 또는 PowerShell을 열고 아래 폴더로 이동합니다.
   ```bash
   cd f:\dev\harmonice\FR-kko
   ```
   - 제공된 자동 실행 스크립트를 실행합니다. (백그라운드에서 백엔드 및 프론트엔드가 모두 실행됩니다.)
   ```powershell
   .\start_persistent.ps1
   ```
   - 패키지 자동 설치 및 서버 기동이 진행됩니다 (소요시간: 약 1~2분).

2. **PC의 IP 주소 확인**
   - 새 명령프롬프트 창을 열고 `ipconfig` 명령어를 입력합니다.
   - **IPv4 주소**를 확인합니다 (예: `192.168.75.201`).

---

## 2️⃣ [태블릿] 네트워크 및 보안 설정

모바일 브라우저(특히 Chrome)는 로컬호스트(`localhost`)가 아닌 IP 환경(`http://192.168...`)으로 접속할 때, 암호화되지 않은 HTTP 연결에서의 카메라(웹캠) 접근을 엄격히 차단합니다. 

원활한 얼굴인식을 위해 **Chrome 브라우저에서 보안 우회 설정(Flag)을 반드시 적용**해야 합니다.

1. **동일 WiFi 접속 확인**
   - 태블릿이 PC와 같은 와이파이(동일 공유기)에 연결되어 있는지 확인합니다.

2. **Chrome 브라우저 실행 및 Flag 설정**
   - 삼성 태블릿에서 Chrome 브라우저를 엽니다.
   - 주소창에 다음 주소를 정확히 입력하고 접속합니다: 
     `chrome://flags/#unsafely-treat-insecure-origin-as-secure`
   - 상단에 나타나는 텍스트 입력창에 PC의 프론트엔드 주소를 입력합니다.
     예: `http://192.168.75.201:3000` (반드시 본인 PC의 IP로 변경!)
   - 입력란 우측의 드롭다운 버튼을 눌러 **Disabled**에서 **Enabled**로 변경합니다.
   - 화면 하단에 나타나는 **Relaunch(다시 시작)** 버튼을 눌러 브라우저를 재시작합니다.

> ⚠️ **주의**: 포트번호(`:3000`)까지 정확히 입력해야 카메라가 동작합니다.

---

## 3️⃣ [태블릿] 시연 진행 흐름

1. **접속 및 환경설정**
   - 브라우저 주소창에 `http://192.168.75.201:3000` (자신의 PC IP)를 입력하여 접속합니다.
   - 우측 상단이나 하단의 `⚙️ 설정` 버튼을 클릭합니다.
   - **백엔드 주소(API URL)**를 `http://192.168.75.201:8000` (포트 8000)으로 수정하고 저장합니다.

2. **사용자 이름 등록**
   - 메인 화면에서 `👤 사용자 등록`을 누릅니다.
   - 이름을 입력하고 등록을 누릅니다.

3. **얼굴 사진 촬영 (개선된 기능)**
   - 이름 등록이 끝나면 **즉시 얼굴 촬영 화면**으로 넘어갑니다.
   - 정면을 바라본 상태에서 `📸 사진 촬영 및 등록` 버튼을 눌러 본인의 얼굴을 백엔드에 학습시킵니다.
   - 성공 메시지가 나타나면 대화창으로 이동합니다. (초기 등록 완료)

4. **얼굴 인식(인증) 테스트**
   - 좌측 상단(또는 하단)의 '뒤로' 버튼을 눌러 다시 메인 홈 화면으로 나옵니다.
   - 이번에는 `🔍 얼굴 인식` 버튼을 클릭합니다.
   - 카메라 렌즈에 얼굴을 맞추고 `📸 촬영 및 인식`을 누릅니다.
   - **등록된 얼굴일 경우**: `인식되었습니다: 이름 (신뢰도: 99%)` 알림과 함께 자동으로 로그인됩니다.
   - **미등록 얼굴일 경우**: 개선된 기능에 의해 `얼굴을 인식하지 못했습니다` 오류 메시지가 뜨며 로그인이 거부됩니다.

## 💡 자주 발생하는 문제(Troubleshooting)

- **"사이트에 연결할 수 없음" 오류 발생 시**
  - PC와 태블릿이 서로 다른 와이파이 망(예: 5G와 2.4G 분리망 등)에 연결되어 있는지 확인하세요.
  - PC의 Windows 방화벽 설정에서 3000포트 및 8000포트의 `인바운드 연결 허용`이 활성화되어 있는지 확인합니다.

- **"카메라 접근 권한이 없습니다" 오류**
  - 브라우저 설정(앱 권한)에서 마이크/카메라 접근이 허용되어 있는지 확인하세요.
  - 위 2단계의 `chrome://flags` 보안 우회 설정이 올바르게 입력되고 Enabled 상태인지 재확인하세요.

- **얼굴 인식 실패율이 높을 때**
  - 태블릿이 위치한 장소의 조명이 충분히 밝은지 점검하세요.
  - 역광 환경에서는 인식이 어려울 수 있습니다.
