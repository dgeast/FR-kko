## 🖥️ PC 준비 (필수)

모든 모바일 디바이스 테스트는 **PC의 백엔드 + 프론트엔드**가 실행 중이어야 합니다.

### Step 1: 터미널 1 - 백엔드 서버 시작

```bash
cd F:\dev\harmonys-claude-integration\projects\facial_recognition\src\backend

# Windows
venv\Scripts\activate

# Mac/Linux
source venv/bin/activate

# 서버 시작
python app.py
```

**성공 메시지:**
```
✅ Uvicorn running on http://0.0.0.0:8000
```

### Step 2: 터미널 2 - 프론트엔드 서버 시작

```bash
cd F:\dev\harmonys-claude-integration\projects\facial_recognition\src\frontend

npm start
```

**성공 메시지:**
```
✅ Compiled successfully!
Webpack compiled with X warning(s)
```

### Step 3: PC의 IP 주소 확인

```bash
# Windows PowerShell / CMD
ipconfig

# Mac Terminal
ifconfig

# Linux Terminal
hostname -I
```

**찾을 것:** IPv4 주소 (예: `192.168.1.100`)

📌 **이 IP를 기억하세요** - 모든 모바일 디바이스에서 사용합니다.

## 📱 샤오미 폰에서 시연

### 📋 사전 체크리스트

- ✅ 샤오미 폰이 PC와 같은 WiFi에 연결
- ✅ 샤오미 폰에 Chrome 또는 기본 브라우저 설치
- ✅ PC에서 백엔드 + 프론트엔드 실행 중
- ✅ PC의 IP 주소 확인 완료

### 🚀 시연 단계

#### 1단계: 브라우저 열기

```
샤오미 폰 → Chrome/브라우저 열기
```

#### 2단계: URL 입력

```
http://192.168.1.100:3000
```

> **주의:** `192.168.1.100`을 **자신의 PC IP**로 변경하세요

#### 3단계: Settings (⚙️) 설정

앱이 로드되면 **우상단 ⚙️ 버튼** 클릭:

```
⚙️ Settings → 백엔드 URL 입력
```

입력값:
```
http://192.168.1.100:8000
```

#### 4단계: 연결 테스트

```
✓ 연결 테스트 버튼 클릭

→ "연결 성공!" 메시지 확인
```

### 📸 시연 시나리오 (샤오미 폰)

#### 시나리오 1️⃣: 사용자 등록

```
홈 화면
  ↓
[👤 사용자 등록] 버튼 클릭
  ↓
이름 입력: "김철수"
  ↓
[등록] 버튼 클릭
  ↓
✅ "사용자 등록 완료!" 메시지
```

#### 시나리오 2️⃣: 얼굴 인식

```
홈 화면
  ↓
[🔍 얼굴 인식] 버튼 클릭
  ↓
💬 "카메라 접근 허용" 팝업 → [허용]
  ↓
📸 Snapshot 버튼 클릭 (자신의 얼굴을 카메라에)
  ↓
✅ "김철수 인식되었습니다" 메시지
  ↓
자동으로 채팅 화면 이동
```

#### 시나리오 3️⃣: AI 대화

```
채팅 화면
  ↓
메시지 입력: "안녕하세요"
  ↓
[전송] 버튼 클릭
  ↓
✅ AI 응답: "안녕하세요! 반갑습니다."
```

#### 시나리오 4️⃣: 사용자 목록

```
홈 화면
  ↓
[👥 사용자 목록] 버튼 클릭
  ↓
등록된 사용자 표시: "김철수"
```

### 🎥 카메라 권한 설정 (샤오미)

얼굴 인식이 작동하지 않으면:

```

  ↓

  ↓
```

## 🔧 문제 해결

### 문제 1️⃣: 모바일에서 페이지가 안 뜨는 경우

```
❌ "이 사이트에 연결할 수 없습니다"
```

**해결책:**

1. PC IP 재확인
```bash
ipconfig
```

2. 올바른 URL 입력 확인
```
http://192.168.1.100:3000  ← 실제 IP로 변경
```

3. 백엔드/프론트엔드 실행 여부 확인
```bash
# PC 터미널에서 확인
python app.py  # 실행 중?
npm start      # 실행 중?
```

4. 방화벽 설정 확인
```
Windows Defender 방화벽 → 앱 허용
Python, Node.js 허용
```

---

### 문제 2️⃣: 백엔드 연결 안됨

```
❌ Settings에서 "연결 실패" 메시지
```

**해결책:**

1. 백엔드 포트 확인
```bash
# 터미널에서 재시작
Ctrl+C (중지)
python app.py (재시작)

✅ "Uvicorn running on http://0.0.0.0:8000" 확인
```

2. URL 형식 확인
```
✅ 올바른: http://192.168.1.100:8000
❌ 잘못된: 192.168.1.100:8000 (http:// 빠짐)
❌ 잘못된: http://localhost:8000 (모바일에서 작동 안함)
```

3. 네트워크 재연결
```
WiFi 끄기 → 다시 켜기
또는
WiFi 비밀번호 재입력
```

### 문제 3️⃣: 얼굴을 인식하지 못함

```
❌ "얼굴을 찾을 수 없습니다"
```

**해결책:**

1. 조명 확인
```
✅ 밝은 환경 (500 럭스 이상)
❌ 어두운 환경, 역광
```

2. 거리 조정
```
✅ 30cm - 60cm 거리 유지
❌ 너무 가까우면: 얼굴이 왜곡됨
❌ 너무 멀면: 인식 안됨
```

3. 각도 확인
```
✅ 정면을 바라봄
❌ 옆모습, 위/아래에서 찍음
```

4. 카메라 권한 확인
```
앱 설정 → 카메라 권한 → [허용]
```

---

### 문제 4️⃣: 느린 응답

```
⏳ 반응이 4-5초 이상 지연
```

**해결책:**

1. WiFi 신호 강도 확인
```
모바일: WiFi 신호 4-5칸 확인
```

2. 네트워크 혼잡 확인
```
가능하면: 2.4GHz → 5GHz로 변경
(라우터 설정)
```

3. 브라우저 캐시 지우기
```
Chrome 설정 → 검색 데이터 삭제
```
