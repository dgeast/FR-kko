## 📋 프로젝트 개요 (Project Overview)

얼굴 인식 기술을 활용하여 사용자를 인증하고, 인증된 사용자와 AI를 통해 날씨, 뉴스, 증권 정보 등을 상호 소통할 수 있는 **PC, 모바일, 태블릿 통합 앱**입니다.

## 🎯 핵심 기능 (Core Features)

### 1️⃣ 얼굴 인증 (Face Authentication)
- 실시간 카메라를 통한 얼굴 감지
- 로컬 DB와의 얼굴 비교 (95% 이상 정확도)
- 3-20명 사용자 등록 및 관리

### 2️⃣ AI 지원 대화 (AI-Powered Chat)
- OpenAI / Claude를 통한 지능형 응답
- 주제별 대화 (날씨, 뉴스, 증권, 일반)
- 외부 API 통합 (OpenWeather, NewsAPI 등)

### 3️⃣ 멀티 플랫폼 지원 (Multi-Platform)
- **데스크톱:** Electron + React (Windows, macOS, Linux)
- **모바일:** Flutter (iOS, Android)
- **웹:** React 기반 관리자 화면

### 4️⃣ 보안 & 개인정보 (Security & Privacy)
- 얼굴정보 AES-256 암호화 저장
- 로컬 저장소 (개인정보보호)
- HTTPS 통신

## 💡 기술 스택 (Tech Stack)

### 백엔드
- **언어:** Python 3.11+
- **프레임워크:** FastAPI
- **얼굴인식:** OpenCV + face_recognition + MediaPipe
- **데이터베이스:** SQLite (로컬), 선택사항: PostgreSQL (확장 시)
- **AI:** OpenAI / Claude API
- **외부 API:** OpenWeather, NewsAPI, Alpha Vantage

### 프론트엔드
- **데스크톱:** React 18+ + Electron 26+
- **모바일:** Flutter 3.16+
- **상태 관리:** Redux (React), Provider (Flutter)
- **통신:** Axios/HTTP

### 개발 도구
- **버전 관리:** Git + GitHub
- **CI/CD:** GitHub Actions
- **컨테이너:** Docker + Docker Compose
- **테스팅:** pytest, Jest, Flutter test
- **모니터링:** ELK Stack (선택사항)
