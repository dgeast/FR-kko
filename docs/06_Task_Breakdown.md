## 6. 완료 기준 (Definition of Done)

각 Task가 다음을 충족할 때 완료로 간주:

1. ✅ **코드 완성:** 모든 기능 구현 완료
2. ✅ **코드 검토:** Code Review 통과
3. ✅ **테스트:** 단위 테스트 80%+ 커버리지
4. ✅ **문서화:** 코드 주석, API 문서 완성
5. ✅ **통합:** 메인 브랜치에 병합
6. ✅ **승인:** 리드/PM 최종 승인
