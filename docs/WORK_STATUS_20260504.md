## 2. 주요 구현 내용

### 📱 얼굴 인식 PWA 핵심 기능 (`pwa/src/components/FaceCapture.js`)
*   **실시간 얼굴 감지**: `react-webcam`과 `face-api.js`를 활용하여 브라우저 환경에서 실시간으로 얼굴을 추적합니다.
*   **자동 인식 모드 (Auto-Capture)**: 
    *   얼굴 인식 신뢰도(Confidence)가 0.8 이상일 때 자동으로 캡처 로직을 실행합니다.
    *   사용자의 수동 조작 없이도 빠르고 정확한 데이터 수집이 가능하도록 설계되었습니다.
*   **캔버스 드로잉 최적화**: 미러 모드(Mirror Mode) 적용 및 `requestAnimationFrame`을 통한 부드러운 UI 렌더링을 구현했습니다.

### ⚡ 저사양 기기 최적화 로직 (`pwa/src/utils/deviceOptimization.js`)
*   **기기 성능 자동 감지**: 
    *   사용자 기기의 RAM(navigator.deviceMemory), CPU 코어 수 등을 분석하여 성능 프로필(High/Medium/Low)을 할당합니다.
*   **Redmi 4A 전용 최적화**:
    *   **감지 간격(Interval) 조정**: 저사양 기기에서는 초당 감지 횟수를 줄여 발열 및 시스템 지연을 방지합니다.
    *   **이미지 스케일링**: 해상도를 낮추어(0.3x) 처리 속도를 향상시켰습니다.
    *   **압축 알고리즘**: 캡처된 이미지를 JPEG 형식으로 압축(Quality: 0.5)하여 메모리 부하를 최소화했습니다.
*   **성능 모니터링**: 실시간 FPS 측정 및 메모리 힙 사이즈 로깅 기능을 포함하여 현장 디버깅이 용이하도록 했습니다.
