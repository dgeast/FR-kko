# 얼굴인식 AI 앱 - 제품 요구사항서 (PRD)

**프로젝트명:** 얼굴인식 AI 지능형 대화 앱
**버전:** 1.0
**작성일:** 2026.04.24
**상태:** 초안

---

## 1. 개요

### 1.1 프로젝트 목표
개인의 얼굴을 인식하여 사용자를 인증하고, 인증된 사용자와 AI를 통해 날씨, 뉴스, 증권 정보 등을 상호 소통할 수 있는 스마트 앱 개발

### 1.2 핵심 가치 제안 (Value Proposition)
- **생체인증:** 비밀번호 없이 얼굴로 즉시 로그인
- **개인화된 상호작용:** 인증된 사용자별 맞춤형 AI 대화
- **실시간 정보 제공:** 인터넷 연결 시 최신 정보 제공

---

## 2. 타겟 사용자 (User Personas)

| 페르소나 | 설명 | 사용 사례 |
|---------|------|---------|
| **일반 사용자** | 가정/사무실 환경의 성인 | 아침에 얼굴인식으로 로그인 후 날씨/뉴스 확인 |
| **관리자** | 앱 설정/사용자 등록 담당 | 새로운 사용자 얼굴정보 등록 |
| **키오스크 운영자** | 공공장소 설치 시 관리자 | 방문객 인증 및 정보 제공 |

---

## 3. 주요 기능 (Core Features)

### 3.1 사용자 관리
- **사용자 등록 (Registration)**
  - 사용자 정보 입력: 이름 또는 가명 (3-20명)
  - 얼굴 캡처: 카메라로 얼굴 촬영 (3-5장)
  - 얼굴정보 저장: 로컬 DB에 인코딩된 얼굴 특성 저장

- **사용자 삭제 (Deletion)**
  - 등록된 사용자 제거
  - 해당 사용자의 얼굴정보 삭제

### 3.2 얼굴 인증 (Face Authentication)
- **실시간 얼굴 감지:** 카메라 입력에서 얼굴 감지
- **얼굴 비교:** 캡처된 얼굴과 DB의 등록된 얼굴 비교
- **인증 결과 표시:**
  - ✅ 인증 성공: 사용자 이름 + 환영 메시지
  - ❌ 미등록: "등록되지 않은 사용자입니다" 메시지

### 3.3 AI 지원 대화 (AI-Powered Chat)
- **조건:** 인증된 사용자만 접근 가능
- **대화 주제:**
  - 🌤️ **날씨 정보:** 현재 위치 기반 기후 정보
  - 📰 **뉴스:** 최신 뉴스 요약 및 카테고리별 정보
  - 📈 **증권 정보:** 주식 시세, 주요 지수
  - 💬 **일반 대화:** AI 챗봇과의 자유로운 상담

- **기능 조건:**
  - 인터넷 없음: 기본 대화만 가능
  - 인터넷 있음: 모든 기능 사용 가능

## 4. 기술 요구사항

### 4.1 플랫폼 지원
- **데스크톱:** Windows, macOS, Linux (Python/Electron)
- **모바일:** iOS, Android (Flutter 또는 React Native)
- **태블릿:** iPad, Android Tablet 지원

### 4.2 핵심 기술 스택
| 계층 | 기술 |
|-----|------|
| 얼굴인식 | OpenCV, face_recognition, MediaPipe |
| AI 대화 | OpenAI API, LLaMA, Claude API |
| 데이터베이스 | SQLite, Firebase (선택사항) |
| 백엔드 | Python FastAPI, Node.js |
| 프론트엔드 | React, Flutter, Electron |
| 외부 API | OpenWeather, NewsAPI, Stock API |

### 4.3 보안 요구사항
- 얼굴정보 로컬 저장 (개인정보보호)
- 암호화된 저장소 사용
- 사용자 개인정보 암호화
- HTTPS 통신 (외부 API 사용 시)

## 5. 데이터 모델 (Data Model)

### 5.1 User (사용자 정보)
```
- user_id: UUID (고유 식별자)
- name: String (이름 또는 가명)
- email: String (선택사항)
- created_at: DateTime
- updated_at: DateTime
- is_active: Boolean
```

### 5.2 FaceData (얼굴 특성 정보)
```
- face_id: UUID
- user_id: UUID (외래키)
- face_encoding: Binary (인코딩된 얼굴 특성)
- capture_date: DateTime
- image_path: String (로컬 저장 경로)
```

### 5.3 ChatHistory (대화 기록)
```
- chat_id: UUID
- user_id: UUID (외래키)
- message: String (사용자 메시지)
- response: String (AI 응답)
- timestamp: DateTime
- topic: String (날씨/뉴스/증권/일반)
```
